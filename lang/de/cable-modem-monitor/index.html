Cable Modem Monitor

Machen Sie die Signaldaten Ihres Modems nutzbar.

DOCSIS-Signalüberwachung für Home Assistant.

10+ Modems 700+ Tests 6 Sprachen

Das Problem

Ihr Kabelmodem erfasst Signalqualitätsdaten — wie stark Ihre Verbindung ist, wie sauber das Signal ist, ob sich Fehler anhäufen. All das ist in der Web-Oberfläche des Modems versteckt. Aber diese Daten sind nicht nutzbar.

  • Keine Historie — Sie sehen nur eine Momentaufnahme, nicht was um 2 Uhr nachts passierte, als Ihre Verbindung abbrach
  • Keine Warnungen — das Modem sagt Ihnen nicht, wenn die Signalqualität nachlässt oder Fehler zunehmen
  • Kein Beweis — wenn Sie Ihren ISP anrufen, ist "das Internet ist langsam" kein Beweis, auf den sie reagieren können
  • Keine Korrelation — Sie können "Zoom ist um 15 Uhr abgestürzt" nicht mit Signalproblemen verbinden, die Sie nicht beobachtet haben

Die Lösung

Cable Modem Monitor holt Signaldaten direkt von Ihrem Modem und bringt sie in Home Assistant. Verfolgen Sie den Verbindungszustand über die Zeit mit historischen Grafiken, die Probleme erfassen, die Sie sonst verpassen würden.

Warum es wichtig ist

Beweise, wenn Sie sie brauchen

Zeigen Sie Ihrem ISP echte Daten — sinkende Signalqualität, steigende Fehlerzahlen. Beweise sind schwerer abzutun als "es fühlt sich langsam an."

Muster über die Zeit

Erfassen Sie die Abbrüche um 2 Uhr nachts und intermittierende Probleme. Probleme, die verschwinden, bevor Sie nachschauen, werden in den Grafiken festgehalten.

Eine Integration, 12+ Modems

Funktioniert mit Arris, Motorola, Netgear und Technicolor. Keine Suche nach einem Parser für Ihr spezielles Modell.

Reaktionen automatisieren

Erhalten Sie Warnungen bei Signalverschlechterung. Starten Sie Ihr Modem automatisch neu, wenn Fehler zunehmen. Bauen Sie Automatisierungen auf echten Daten auf.
